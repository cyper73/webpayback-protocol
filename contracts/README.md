# 📝 WebPaybackToken (WPT) - Smart Contracts

This folder contains the Solidity smart contracts for the **WebPaybackToken (WPT)** project on Polygon.

## Main Contracts

- **WebPaybackToken.sol**  
  - ERC20 token contract for WPT with creator fee mechanism (3% by default).
  - **Deployed address (Polygon):** [`0x9077051D318b614F915E8A07861090856FDEC91e`](https://polygonscan.com/address/0x9077051D318b614F915E8A07861090856FDEC91e)
