### 📋 Installation Instructions:

1. **Extract Archive**:
   ```bash
   tar -xzf webpayback-complete-final-20250727-1845.tar.gz
   cd webpayback-complete-final-20250727-1845
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Environment Setup**:
