```bash

# Navigate to project

# Initialize git

git add .

```

### 3. Connect to GitHub
```bash
# Add remote (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/webpayback-protocol.git

# Push to GitHub
git branch -M main
git push -u origin main
```
